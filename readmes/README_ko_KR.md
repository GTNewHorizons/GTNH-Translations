# GTNH Translations

여기에서는 한국어를 포함한 모든 GTNH 번역 프로젝트가 관리되고 있습니다.

## 사용법

1. [Releases](https://github.com/GTNewHorizons/GTNH-Translations/releases) 에서 ko_KR Latest를 선택
2. 다운로드 및 압축을 풀고 .minecraft 폴더내부에 집어넣기

## 피드백 및 제안
